> Setting Up The Structure

We will be developing a dapp against the live blockchain. A dapp is an application that runs on a decentralized peer to peer network that is backed by a smart contract written in Solidity. It will then run on Ganache which automatically creates 10 test accounts preloading them with 100 units of fake ether.
